🚀 DEPLOY RÁPIDO (3 MINUTOS)
═══════════════════════════════

OPÇÃO 1 - Drag & Drop (Mais Fácil):
────────────────────────────────────
1. Vá em: https://app.netlify.com
2. Faça login (ou crie conta grátis)
3. Clique em "Add new site" → "Deploy manually"
4. Arraste a pasta "dash_public" inteira
5. Pronto! Você receberá um link tipo: https://seu-site.netlify.app

OPÇÃO 2 - Via CLI:
──────────────────
1. Abra PowerShell nesta pasta (dash_public)
2. Execute:
   npm install -g netlify-cli
   netlify login
   netlify deploy --prod

OPÇÃO 3 - Via GitHub:
─────────────────────
1. Crie um repositório no GitHub
2. Faça push desta pasta
3. No Netlify, conecte o repositório
4. Deploy automático a cada commit!

📖 Guia Detalhado: Veja DEPLOY_NETLIFY.md


🧪 TESTAR ANTES DO DEPLOY
═════════════════════════

1. Abra PowerShell nesta pasta (dash_public)
2. Execute: python -m http.server 8000
3. Abra: http://localhost:8000
4. Verifique:
   ✅ Cards aparecem nas 3 colunas
   ✅ Console sem erros (F12)
   ✅ Real-time funciona


🎯 O QUE FAZ O DASHBOARD
════════════════════════

📊 3 Colunas em Tempo Real:
   • Disponível ──────── Pets prontos para adoção
   • Em Processo ──────── Pets com adoção em andamento
   • Adotado Hoje ────── Pets adotados no dia atual

⚡ Funcionalidades:
   • Atualização em tempo real via Supabase Realtime
   • Cards se movem automaticamente entre colunas
   • Responsivo (mobile, tablet, desktop)
   • Seguro (apenas dados públicos, RLS ativo)
   • Design idêntico ao app Vi.Ca


🔒 SEGURANÇA
════════════

✅ Banco de dados protegido:
   • Usa apenas chave ANON (pública)

✅ Dados NÃO expostos:
